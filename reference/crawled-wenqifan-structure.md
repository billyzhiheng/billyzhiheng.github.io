# 参考抓取摘要：Wenqi Fan — Workshops 页面结构

来源：<https://wenqifan03.github.io/workshops.html>

## 页面划分（两栏）

| 左栏（窄） | 右栏（主内容） |
|------------|----------------|
| 站点/人名标题 | 页面大标题（与当前页对应） |
| **Research** 分组下若干内链 | 二级标题 + 条目列表 |
| **Professional Activities** | 每条：作者、标题、会议/出处、年份、外链 |
| **Experience** | |
| **Links**（Scholar、DBLP 等） | |

## Demo 中的对应实现

- 用 **CSS 两栏布局**（左固定宽度侧栏 + 右主栏）复现「左导航分组 + 右正文」。  
- 右栏正文采用 Wikipedia 式排版（导语、TOC、H2/H3、附录），形成风格融合。
