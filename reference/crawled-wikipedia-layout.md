# 参考抓取摘要：Wikipedia Manual of Style / Layout

来源：<https://en.wikipedia.org/wiki/Wikipedia:Manual_of_Style/Layout>  
抓取说明：正文为 Markdown 化摘要，用于本地实现「维基式」视觉与区块顺序，非完整 HTML 镜像。

## 页面元素顺序（节选）

- **文前**：导航模板、图片、维护模板等  
- **正文**：Body → 目录 (TOC) → **Lead（导语/引言）**  
- **附录**：External links → Further reading → Notes and references → See also → Works or publications  

## Body 与标题

- 二级标题 `== Heading 2 ==`，三级 `=== Heading 3 ===`，不跳级。  
- 标题与正文之间留白克制；段落首行不缩进。  
- 列表用于参考文献、延伸阅读、外部链接等附录区。

## 标准附录（页脚区）

- **See also**：项目符号列表，首字母大写。  
- **References / Notes**：引用脚注或文献列表。  
- **Further reading**：可选书目列表。  
- **External links**：带简短说明的推荐外链列表。

## Specialized layout

独立列表、讨论页等有单独版式；部分主题（影视、医学等）有专门 MoS 版式说明。
